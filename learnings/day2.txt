Dynamic Array v/s Dynamically Allocated Arrays

A dynamically allocated array is allocated memory size on runtime,
but after allocation, it is a static array at the end.
whereas
A dyamic array changes it memory size whenever it runs out of space
It can also reduce its memory size if needed
